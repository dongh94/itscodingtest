# 시각

## 문제
* 정수 N이 입력되면 00시00분00초 부터 N시 59분 59초까지의 모든 시각중에서 3이 하나라도 포함되는 모든 경우의 수를 구하는 프로그램을 작성하시오.
  * 00시 00분 13초
  * 00시 13분 30초 등..
* 반면에 다음은 3이 하나도 포함되어 있지 않으므로 세면 안 되는 시각이다.
  * 00시 02분 55초
  * 01시 27분 45초

## 입력
* N (0<=N<=23)

## 출력
* 00시 00분 00초부터 N시 59분 59초 까지의 모든 시각중에서 3이 하나라도 포함된 모든 경우의 수를 출력하시오
